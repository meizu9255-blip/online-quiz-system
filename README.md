🎓 QuizKZ — Онлайн тест тапсыру жүйесі
QuizKZ — бұл студенттердің білімін тексеруге арналған заманауи веб-қосымша. Жоба React (Frontend) және Node.js (Backend) технологиялары негізінде жасалған.

✨ Негізгі функциялары
Авторизация: Әр пайдаланушы өз атымен тіркеліп, жүйеге кіре алады.

Динамикалық тесттер: 5 түрлі бағыт (HTML, CSS, JS, Желілер, Қауіпсіздік) бойынша тест тапсыру.

Нақты уақыттағы нәтиже: Тест біткен бойда балл есептеледі және қатемен жұмыс жасау үшін әр сұрақтың аналитикасы шығады.

Рейтинг жүйесі (Leaderboard): Барлық пайдаланушылардың үздік нәтижелері негізінде түзілетін жалпы рейтинг.

Жеке профиль: Пайдаланушының жеке статистикасы (орташа балл, рекорд, тапсырылған тесттер саны).

Түнгі режим: Көзді шаршатпау үшін Dark Mode функциясы қарастырылған.

🛠 Технологиялық стек
Frontend: React.js, React Router, Axios, CSS3 (Modern Flexbox/Grid).

Backend: Node.js, Express.js.

Дерекқор: JSON-based database (db.json) — деректерді файлдық жүйеде сақтау.

🚀 Жобаны іске қосу нұсқаулығы
1. Репозиторийді жүктеу немесе көшіру
Bash
git clone [репозиторий-сілтемесі]
cd online-quiz-system
2. Бэкендті іске қосу
Bash
cd backend
npm install
npm run dev
Сервер келесі мекенжайда қосылады: http://localhost:5000

3. Фронтендті іске қосу
Жаңа терминал ашып:

Bash
cd frontend
npm install
npm run dev
Сайт келесі мекенжайда ашылады: http://localhost:5173

📁 Жоба құрылымы
/backend — API сервер және деректерді сақтау (server.js, db.json).

/frontend — Пайдаланушы интерфейсі және логика (React components, pages).

/data — Тест сұрақтары мен нұсқалары
