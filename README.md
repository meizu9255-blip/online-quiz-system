# 🎓 QuizKZ - Инновациялық Онлайн Тестілеу Жүйесі

**QuizKZ** — оқушылар мен студенттердің білімін тексеруге, қателермен жұмыс жасауға және прогресті бақылауға арналған толыққанды (Full-Stack) веб-қасымша. 

Жоба "КМ04 Web-сайтты жобалау және үздіксіз жұмыс істеуін қамтамасыз ету" пәні бойынша **Курстық жобаға** арналып, ең жоғарғы сапа стандарттарына (UX/UI, REST API оңтайландыру, мазмұндық архитектура) сай дайындалған.

---

## 📸 Скриншоттар мен Интерфейс
*(Суреттерді осында жүктеп, мына жердегі сілтемелерді орната аласыз)*
1. **Басты бет (Dashboard)** - Тесттер каталогы және статистика.
2. **Профиль (Radar Chart)** - Оқушының динамикалық аналитикасы және Рейтингі.
3. **Smart Quiz** - Интеллектуалды қателермен жұмыс.

---

## 🛠 Технологиялық Стек (Tech Stack)

### Front-end (Клиенттік бөлік)
- **React.js** (Functional Components, React Hooks: `useState`, `useEffect`)
- **React Router DOM** (Көпбеттік навигация)
- **Recharts** (Профильдегі динамикалық Radar Chart графиктері)
- **CSS3** (Tailwind қолданбай толыққанды Custom Modern UI CSS, Responsive Media Queries)

### Back-end (Серверлік бөлік)
- **Node.js** & **Express.js** (Сервер логикасы, бағыттаулар)
- **PostgreSQL** (Реляциялық дерекқор, `pg` драйвері)

### Автоматтандыру және Құралдар
- **Git / GitHub** (Кодты басқару)
- **Vite** (Жылдам минификацияланған Front build-тар)
- **Render.com** (Реттелген Cloud Deployment)
- Семантикалық HTML5 (`<header>`, `<main>`, `<article>`, `<section>`, `<nav>`)

## 📡 REST API Құжаттамасы

Сервер мекенжайы (Base URL): `https://online-quiz-system-ufwp.onrender.com/api`

| Әдіс (Method) | Эндпоинт (Endpoint) | Сипаттамасы (Description) |
| --- | --- | --- |
| `GET` | `/quizzes` | Базадағы барлық қолжетімді тесттерді алу. |
| `POST` | `/quizzes` | **(Админ)** Жаңа тест жасау. |
| `PUT` | `/quizzes/:id` | **(Админ)** Бар тесттің ақпаратын (сұрақтарын) жаңарту. |
| `DELETE` | `/quizzes/:id` | **(Админ)** Тестті базадан біржола өшіру. |
| `POST` | `/results` | Оқушының тест нәтижесін және қателерін (`mistakes`) жіберу. |
| `GET` | `/results` | Барлық пайдаланушылардың ортақ нәтижелерін алу (Рейтингке). |
| `GET` | `/analytics/:username` | Оқушының күрделі аналитикалық мәліметтерін (Radar дата) сұрау. |
| `GET` | `/smart-quiz/:username` | "Қателермен жұмыс" үшін (Mistakes кестесінен) жеке генерацияланған тест алу. |

## 🚀 Жобаны Орнату және Іске Қосу (Local Setup)

Жобаны өз санатыңызда (localhost) көтеру үшін мына әрекеттерді орындаңыз.

### 1. Репозиторийді жүктеу
```bash
git clone https://github.com/meizu9255-blip/online-quiz-system.git
cd online-quiz-system
```

### 2. Back-end модулін іске қосу
```bash
cd backend
npm install
# Серверді бастау (ол автоматты түрде Дерекқордағы 5 кестені құрып береді!)
npm start 
```

### 3. Front-end модулін іске қосу
Келесі жаңа терминал терезесінде:
```bash
cd frontend
npm install
npm run dev
```
Сайт автоматты түрде бразуерде `http://localhost:5173` мекен-жайында ашылады.

## 🔑 Админ панеліне кіру
Админ панеліне сілтеме: `/admin`. Бет тек жүйелік әкімшілерге арналған. 
- Тіркелу және жүйеге кіру кезінде **Пайдаланушы аты:** `admin` (кіші әріппен) немесе `Admin` деп енгізсеңіз, сізге тест қосуға, өшіруге және өзгертуге мүмкіндік беретін "Админ Панелі" мәзірі ашылады.

## 💡 Ерекше мүмкіндіктер мен құрылым

- **Data Driven Radar Chart:** Әр оқушының қаншалықты қай саладан (Web, Программалау, Networks) қандай қабілеті бар екені профилінде нақты графикпен көрсетіледі.
- **Smart Quiz (Қателермен жұмыс):** Басқа сайттардан айырмашылығы — жүйе тек нәтижені сақтап қоймайды, ол `mistakes` кестесі арқылы оқушының қай сұрақтан қате жібергенін іліп алып, болашақта оған сол сұрақтарды қайта тапсыртуға бағыттайды.
- **Auto DB Generator:** Жобаның `server.js` ішіндегі `initDatabase` функциясы арқасында дерекқор алғашқы қосылым кезінде 5 басты кестені (`users, quizzes, results, leaderboard, mistakes`) өз бетінше автоматты түрде құрып алады. Бұл бөгде адамға орнатуды (Deploy) өте жеңілдетеді.
