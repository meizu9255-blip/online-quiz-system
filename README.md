# QuizKZ - Онлайн тест тапсыру және нәтижесін шығару жүйесі

## Жоба туралы
QuizKZ - бұл HTML, CSS және JavaScript технологияларымен жасалған 
онлайн тест тапсыру веб-қосымшасы. Пайдаланушылар тіркеліп, 
әртүрлі тақырыптар бойынша тесттер тапсырып, нәтижелерін көре алады.

## Қолданылған технологиялар
- HTML5 - семантикалық тегтер (header, nav, main, section, footer)
- CSS3 - Flexbox, Grid, CSS Variables, Media Queries, Animations
- JavaScript (ES6+) - DOM manipulation, Event handling, localStorage API
- localStorage - деректерді сақтау (пайдаланушылар, тесттер, нәтижелер)
